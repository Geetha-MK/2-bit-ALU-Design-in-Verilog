# 🔢 2-bit ALU Design in Verilog

## 📌 Project Description
This project implements a **2-bit Arithmetic Logic Unit (ALU)** using Verilog HDL.  
The ALU performs basic arithmetic and logical operations based on a 3-bit select signal (`sel`).  
The design is verified using a testbench and simulated using **EDA Playground**.

---

## ⚙️ Operations Performed

| Select (sel) | Operation        |
|--------------|------------------|
| 000          | Addition         |
| 001          | Subtraction      |
| 010          | AND              |
| 011          | OR               |
| 100          | XOR              |
